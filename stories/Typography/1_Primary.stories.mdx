### Please use the following font stack:
This ensures that Arabic letters are displayed correctly. and a good fallback font is used while loading the Ubuntu font.
<Source
  language='css'
  dark
  format={false}
  code={dedent`
    font-family: 'Ubuntu', 'Noto Kufi Arabic', 'Geeza Pro', -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Oxygen', 'Cantarell', 'Fira Sans', 'Droid Sans', 'Helvetica Neue', sans-serif;
  `}
/>

1. Ubuntu = This is our **Volt default font**. (It includes latin and arabic characters.)
<br /><br />
2. Noto Kufi **Arabic** = A Noto font from Google (maybe it's installed)
3. Geeza Pro = macOS Arabic Default
<br /><br />
4. -apple-system, BlinkMac**SystemFont** = macOS Default
5. Helvetica Neue = Old macOS Default
6. Segoe UI = Windows Default
7. Roboto = Android Default
8. Fira Sans = Firefox OS Default
9. Oxygen, Cantarell, Droid Sans = Linux
<br /><br />
10. sans-serif = **Fallback**
